# 設備日常点検記録アプリ

## 概要
工場や施設の設備点検を簡単に記録・保存できるC#コンソールアプリケーションです。
点検結果（OK/NG）とコメントを記録し、CSV形式で保存します。

## 必要な環境
- .NET 8.0 SDK

## 実行方法

### 1. アプリケーションの実行
```bash
dotnet run
```

### 2. 使い方
1. 表示される設備リストから番号を選択、または「その他」を選択して設備名を入力
2. 点検結果を入力（OK または NG）
3. コメントを入力（任意、Enterキーでスキップ可能）
4. 続けて点検を記録する場合は「y」、終了する場合は「n」を入力

## 出力ファイル
- ファイル名: `inspection_results_YYYYMMDD.csv`
  - 例: `inspection_results_20260212.csv`
- 形式: CSV（日時,設備名,点検結果,コメント）
- 文字コード: UTF-8

## ファイル構成
- `Program.cs` - メインプログラム
- `sample_app.csproj` - プロジェクトファイル
- `README.md` - このファイル

## 点検対象設備
デフォルトで以下の設備が登録されています：
1. ポンプA
2. ポンプB
3. コンプレッサー
4. ボイラー
5. 冷却塔
6. 電気盤
7. 配管システム
8. 換気扇

設備を追加する場合は、Program.cs の `EquipmentList` 配列を編集してください。
